个人解决方案

使用 Phoenix (A lightweight macOS/OS X window and app manager scriptable with JavaScript)

应用切换

使用快捷键切换常用应用, IDE, 浏览器, 通讯工具, NOTE, MAIL...

  • ALT + 1 = IDEA
  • ALT + q => Chrome
  • ALT + , => Mail
  • ALT + 3 => Spotify

窗口管理

CMD+CTRL+F 全屏
CMD+CTRL+H 左半屏
CMD+CTRL+L 右半屏
CMD+CTRL+K 上半屏 
CMD+CTRL+J 下半屏
CMD+CTRL+G 居中

CMD+CTRL+= 等比增大 
CMD+CTRL+- 等比缩小

CMD+CTRL+P 移动当前窗口到上一个工作空间 
CMD+CTRL+N 移动当前窗口到下一个工作空间

# 双显示器
ALT+SHIFT+H 移动当前窗口到上一个显示器
ALT+SHIFT+L 移动当前窗口到下一个显示器

# 同工作空间窗口切换
ALT+J 下一个
ALT+K 上一个

# 光标
ALT+M 光标窗口居中
