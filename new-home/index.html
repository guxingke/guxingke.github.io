博客的新家

Published: 2019-07-20

blog

从现在这个时间点来看, 我的个人小站几年内应该不会再变来变去了, 会在 github pages 定居较长的一段时间了.

回想起来, 个人博客小站的历程算是比较丰富了, 稍记一二.

启蒙

当时许是看了什么不写博客的技术人不是好厨子类似的鸡汤, 开始了博客之旅, 一开始在 CSDN, OSCHINA 两站写了一些笔记, 现在仍能找得到相关痕迹.

折腾开始

折腾的最开始, 印象不太深刻, 应该是在某云上搭建 wordpress 开始, 仅限于体验, 后服务过期, 与 wordpress 就此别过

开发自己的 blog

这时候应该大三了, 中间折腾过 typecho , 感觉还不错, 但我当时主要研习 Java , typecho 由 PHP 构建, 而 PHP 的风评向来不佳, 便萌生了自研的想法, 当时行动力还不 错, 很快模仿 typecho 的个人实现 typecho4j 开发完毕, 部署在阿X云上.

具体停用 typecho4j 的缘由, 已记不得了, 反正最后没用上.

ghost

彼时 ghost 还只是单纯的博客程序, 用了一段时间, 毕竟都是要服务器的, 后来因为某次云服务忘记续费, 数据丢失, 与 ghost 的交情止步于此.

hexo & hugo

跟上静态 blog 之风, 使用 hexo, hugo 一段时间. 但因为多终端, 都需要装环境, 下载 theme, 总有不便. 弃之.

bitcron

bitcron 是 macos 平台下一款优秀的 Markdown 编辑器的 MarkEditor 同工作室作品, 功 能还算比较强大, 做个 blog 轻而易举, 但因为基于 Dropbox 等类似云盘, 同步时机不可 控. 且时常抽风, 访问结果不符合预期, 弃之.

pages

就是当前的新家了,不到 120 行核心代码, theme 来自 vitrun, 作了微调.

小结

几年来, 有效的产出没有几篇, 倒是在工具本身上折腾, 有违初衷, 此次定居, 当慎而易之, 专心输出为上.
